# 🎨 Digitale Drukkerij & Reclamebureau Website

Dit is de broncode voor de informatieve website van ons reclamebureau. De website is gebouwd met een focus op snelheid, eenvoud en duidelijkheid.

## 🚀 Over de website
Deze website dient als digitaal visitekaartje voor onze drukkerij. We hebben gekozen voor een "content-first" benadering:
- **Simpel:** Geen overbodige toeters en bellen.
- **Informatief:** Focus op onze drie kerntaken (Stickers, Ontwerp, Drukwerk).
- **Geen directe verkoop:** Bezoekers worden aangemoedigd om persoonlijk contact op te nemen voor advies op maat.

## 🛠️ Gebruikte Technieken
- **HTML5:** Voor de structuur van de pagina.
- **CSS3:** Voor een modern en responsive design (werkt op mobiel, tablet en desktop).
- **GitHub Pages:** Voor de hosting van de website.

## 📁 Bestandsstructuur
- `index.html`: Bevat zowel de structuur als de opmaak (CSS) van de website.

## ✍️ Aanpassingen maken
Wil je de tekst of contactgegevens aanpassen?
1. Open het bestand `index.html`.
2. Zoek de tekst die je wilt veranderen (bijvoorbeeld het telefoonnummer of adres).
3. Sla de wijzigingen op via 'Commit changes'. De website wordt automatisch bijgewerkt.
